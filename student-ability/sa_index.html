综合能力诊断

COMPREHENSIVE ABILITY DIAGNOSIS

  • 采集学习过程大数据
  • 多维度解读八项能力
  • 个性化、可视化表征
  • 发现问题，解决问题

本系统旨在诊断学习者的综合能力，通过采集学习者学习的全过程行为数据，对学习者的认知、思维、调控信息加工、自主学习、社会交往、问题解决和创新八个方面进行个性化诊断，发现学习者存在的问题，并从相应角度为学习者提出具体的学习建议，促进学习者综合能力的提升。

学习困难评估

LEARNING DISABILITY ASSESSMENT

  • 尺有所短，寸有所长
  • 博采众长，相得益彰
  • 分析评估三大困难
  • 提供对应干预建议

本系统旨在评估学习者的学习困难，包括暂时性困难、能力型困难和动力型困难，发现学习者在学习兴趣、学习动机、学习态度、知识掌握以及语言、阅读、书写、数学等方面的具体困难，让学习者了解影响自己学习的真正因素，并依据对应的学习建议不断改进学习过程、提高学习效率。

学习绩效分析

LEARNING PERFORMANCE ANALYSIS

  • 考试作业点滴分析
  • 学习伙伴精准推荐
  • 历次错题详尽收录
  • 百尺竿头更进一步

本系统旨在分析学习者的学习绩效，由考试和作业两部分组成，通过分析各学科的历次考试和作业的班级平均成绩和个人成绩变化趋势，包括对考试和作业中的容易、中等、困难三类题型的分析，可视化呈现出每位学习者学习绩效的综合变化，同时实现对个性化学习同伴的智能推荐。

心理画像

PSYCHOLOGICAL PROFILING

  • 你知道真正的自己吗？
  • 七大维度全面剖析
  • 我比你更了解你自己！
  • 个性优势发掘增强

本系统旨在分析学习者的心理画像，通过对学习者的学习动机、学习风格、策略特征、情绪特征、人格特征成就归因和自我效能感的分析，描绘出学习者的学习心理画像，并提出学习者在这七方面需要保持和注意的问题，让教师、家长和学习者本人都更真实详细地了解每位学习者的特点。
